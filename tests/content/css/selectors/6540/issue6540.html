Issue 6540: Dynamically update selector groups when framed elements change

Steps to reproduce

  1. Open Firebug
  2. Switch to the CSS panel
  3. Inside the Selectors side panel click the Try a selector... field
  4. Enter div and hit Enter
    ⇒ A <div> should be listed inside the Selectors side panel.
  5. Click the Add <div> button above
    ⇒ A blue <div> will be added above the button.

Expected result

  • The Selectors panel should now list two <div>s.
